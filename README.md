# fetch-cloudnestra-from-tmdb_imdb
This script fetches the cloudnestra url from the imdb id or tmdb id. examples: 
- http://127.0.0.1:4343/fetch?tmdb=1007757
- http://127.0.0.1:4343/fetch?imdb=tt2953050

  You can also put custom urls

- http://127.0.0.1:4343/fetch?url=https://ythd.org/embed/tt2953050

  This script mainly targets sites with Iframe loading
This script runs on port 4343
